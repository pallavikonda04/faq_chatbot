# 🤖 FAQ Chatbot

A Semantic FAQ Chatbot built using **Python** and **Streamlit** that answers AI and Data Science related questions using **Sentence Transformers** and **Cosine Similarity**.

## 🚀 Features

- Semantic Search using Sentence Transformers
- Cosine Similarity for finding the best FAQ
- Token Count using Tiktoken
- Interactive Streamlit Web Interface
- AI & Data Science Knowledge Base
- Displays Similarity Score
- User-friendly Interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Sentence Transformers
- Scikit-Learn
- Pandas
- NumPy
- Tiktoken
